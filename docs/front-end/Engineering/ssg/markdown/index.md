# markdown
## 常用
- 删除线（Strikethrough）
```markdown
~~The world is flat.~~
```
- 表格
```markdown
| Syntax      | Description |
| ----------- | ----------- |
| Header      | Title       |
| Paragraph   | Text        |
```
- 代码折叠
```markdown
<details :closed="closed">
    <summary class="">{{ title }}</summary>
    <slot></slot>
  </details>
```
