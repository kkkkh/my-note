## Vscode
### 快捷键
- 所有折叠：Ctrl + k & Ctrl + 0
- 所有展开：Ctrl + k & Ctrl + j
- code /path/to/my/project 的方式打开 VSCode
```bash
View -> Command Palette / 查看 -> 命令面板 / ctrl + shift + p 
输入install code -> 选择提示的Shell Command: Install 'code' command in PATH 回车
命令行 code /path/to/my/project 的方式打开 VSCode
```
